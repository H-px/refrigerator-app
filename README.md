# Refrigerator App 🧊

A mobile-first, glassmorphism fridge inventory manager.

## ✨ Features

- **PWA Support**: Install to your home screen and use offline.
- **Smart Search**: Quickly find items in your fridge, freezer, or pantry.
- **Data Portability**: Easily export and import your inventory data.
- **Expiration Tracking**: Color-coded status for fresh, warning, and expired items.
- **Dark/Light Mode**: Seamlessly switch themes.
- **LocalStorage**: Your data stays on your device.

Built with React (CDN) and Love.
